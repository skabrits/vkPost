# Ынструкция

## Шаг 1. Разово создать токен сообщества

Нужно быть админом группы.

1. Зайди на страницу сообщества с браузера.
2. Нажми «Управление».
3. Перейди в «Настройки → Дополнительно → Работа с API» (или «Использование API» / «API usage») → вкладка «Ключи доступа» / «Access tokens».
4. Нажми «Создать ключ» (Create token).
5. Отметь права, минимум:
    - «Управление сообществом» / «Allow the application to control community»;
    - «Доступ к сообщениям», «Фотографии» — если планируешь постить с картинками.
6. Подтверди через SMS/приложение.
7. Скопируй получившийся community access token.
8. Запомни ID группы (цифры в URL: vk.com/club12345678 → group_id = 12345678).

## Шаг 2. Положить токен на сервер

На машине, где будет крутиться бот, сделай файл .env рядом со скриптом:

```env
VK_GROUP_ID=12345678
VK_GROUP_TOKEN=vk1.a.XXXXXXXXXXXXXXXXXXXXXXXXXXXX
```

## USAGE

```bash
python vk_group_bot.py --message "Привет, это бот"
python vk_group_bot.py --message-file post.txt
```
